Hungry Belly

Description

So, it is lunch time. You and your friends would like to meet to have lunch, but do not know which resturants will work best. Now, with Hungry Belly, you can find resturants located between all your friends. After finding the centralized resturants, you can send a text message through Hungry Belly and enjoy lunch.

Technologies used

  • Ruby on Rails
  • PostgreSQL
  • Google Maps API
  • Twilio Gem
  • Geocoder Gem
  • Yelp Gem
  • Twitter Bootstrap
  • HTML5
  • CSS3
