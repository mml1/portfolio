Remind Me

Description

Created over the course of two days, Remind Me allows users to create reminders for tasks at specified times or certain intervals.

Technologies used

  • Ruby on Rails
  • SQLite
  • Rufus Scheduler Gem
  • Twitter Bootstrap
  • HTML5
  • CSS3
