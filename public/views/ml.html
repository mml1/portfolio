Memory Lane

Description

A memory game where you have to reveal two matching cards to earn a point. See your stats in bar format once the game is over. Want to know if you made it to the top ten? Click on past records.

I was inspired to make Memory Lane after seeing Seniors having difficulties playing a similar game using playing cards. They were trying to flip a card over but could not due to loss of range of motion. I made this game, so people with similar impairments can play. Memory Lane was intended to be played on a tablet but can be played with a mouse. Enjoy!
