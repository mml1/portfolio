Memory Lane

Description

A memory game where you have to reveal two matching cards to earn a point. See your stats in bar format once the game is over. Want to know if you made it to the top ten? Click on past records.

Technologies used

  • AngularJS
  • Node.js
  • Express.js
  • Mongoose
  • MongoDB
  • Twitter Bootstrap
  • HTML5
  • CSS3
