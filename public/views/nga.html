Shake Quake


Description

Shake Quake is an Earthquake visualizer intended to show children earthquakes from different decades and of different magnitudes. Created over the course of 24 hours, Shake Quake uses MapBox to display earthquake data from the Humanitarian Data Exchange.

Technologies used

  • MapBox GL
  • jQuery
  • Vanilla JS
  • HTML5
  • CSS3
