Recycle Me

Description

Not sure how to recycle an item? Ask Alexa! Recyle Me is an application for the Amazon Echo to help a person recycle. In additon, you can check how full your recyling bin is through a sensor created using Arduino Edison and the Grove starter kit. Iot Cities hackathon was the first time I touched hardware.

Technologies used

  • Amazon Alexa
  • NodeJS
  • Arduino Edison Board
  • Grove Starter Kit
  • JSON
